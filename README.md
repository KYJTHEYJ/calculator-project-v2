## 프로그램 소개
### - **Step3 완성본 기준**

1. 프로그램 실행시 2개의 숫자를 차례대로 입력 받습니다
    - 숫자 형식이 아닌 키워드가 입력시 경고와 함께 재입력을 받습니다
<img width="233" height="32" alt="img_1" src="https://github.com/user-attachments/assets/0d985077-35ea-465d-b00c-6e2cb76ce479" />


<img width="223" height="51" alt="img_2" src="https://github.com/user-attachments/assets/1b34c20c-3131-4b54-90d8-dc49ae15422b" />


2. 사칙연산 기호를 입력 받습니다
   
   - ( +, -, *, / ) 사칙 연산 기호만 입력 받을 수 있습니다
   - 기타 기호나 키워드가 잘못된 입력시 경고와 함께 재입력을 받습니다
   

<img width="271" height="23" alt="img_3" src="https://github.com/user-attachments/assets/bf907f41-a2ec-4263-b68d-02efb2d1be5d" />


3. 사칙연산 기호를 입력 받으면 결과가 출력되고 다른 기능에 대한 안내가 출력됩니다
        
   - 기본적으로 아래 사항에 입력이 올바르지 않거나 결과가 없는 경우에 대한 안내도 제공되고 있습니다
